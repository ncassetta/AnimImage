def animimage.set_debug (   f)

Turn on/off the debug mode.


When you import the file debug mode is disabled, if the user turns it on the various methods will print additional informations on the console. /param f True or False.

◆ set_log()

def animimage.set_log (   f)

Turn on/off the generation of a log file of the GIFDecoder.


When you import the file the log is disabled; if the user turns it on the GIFDecoder.decode() method will print detailed info on the decoding process into a GIFDecoder.log file in the same folder of this file. /param f True or False.
